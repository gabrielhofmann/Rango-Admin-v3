Documentação

1. Estrutura:

    a) Páginas e componentes:
        
        Arquivos dentro da pasta pages representam cada página do site e são utilizados para gerenciar rotas em main.jsx.
        
        Páginas podem conter componentes, estes estarão na pasta components.

    b) Estilização:

        O arquivo /src/base.scss contém os estilos globais ou reutilizáveis do site, contando também com variáveis para padronização.

2. Chamadas API:

    Todas as chamdas API são executadas através da classe Services, localizada em /src/services.js.
